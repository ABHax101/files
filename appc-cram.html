Exponent Rules
Core Rules
productaᵐ · aⁿ = aᵐ⁺ⁿ
quotientaᵐ / aⁿ = aᵐ⁻ⁿ
power of power(aᵐ)ⁿ = aᵐⁿ
zero exponenta⁰ = 1   (a ≠ 0)
negative expa⁻ⁿ = 1/aⁿ
fractional expaᵐ/ⁿ = ⁿ√(aᵐ)
specificallya¹/ⁿ = ⁿ√a
a⁻ⁿ moves to the denominator — it does NOT make the base negative. (-a)ⁿ and -(aⁿ) are different things.
Distribution Rules
product dist.(ab)ⁿ = aⁿ · bⁿ
quotient dist.(a/b)ⁿ = aⁿ / bⁿ
neg base (even)(-a)ⁿ → positive
neg base (odd)(-a)ⁿ → negative
1 raised to any1ⁿ = 1
any raised to 1a¹ = a
You CANNOT distribute an exponent over addition: (a+b)ⁿ ≠ aⁿ + bⁿ. This is probably the most common algebra error on this exam.

The horizontal asymptote of a·bˣ + k is always y = k (the vertical shift). Watch for this on MCQs.
Logarithm Rules
Definition & Core Rules
definitionlogₐ(x) = y  ↔  aʸ = x
productlog(xy) = log x + log y
quotientlog(x/y) = log x − log y
powerlog(xⁿ) = n · log x
change of baselogₐ(x) = ln x / ln a
alsologₐ(x) = log x / log a
log(x + y) ≠ log x + log y. You can ONLY split/combine logs across multiplication and division, never addition or subtraction inside the argument.
Special Values & Inverses
log of 1logₐ(1) = 0
log of baselogₐ(a) = 1
inverse #1logₐ(aˣ) = x
inverse #2a^(logₐ x) = x
ln of eln(e) = 1
ln of 1ln(1) = 0
domainx > 0 strictly
Domain of log is always x > 0. If the argument is an expression, set it > 0 and solve. This is a very common trap question.
Logarithmic Functions — Key Behaviors
f(x) = logₐ(x)
domainx > 0
rangeall real numbers
x-intercept(1, 0) always
asymptotevertical: x = 0
b > 1increasing
0 < b < 1decreasing
Log and exponential are inverses — their graphs are reflections over y = x. Their domain and range are swapped.
Solving Exponential & Log Equations

same basebᵐ = bⁿ → m = n

Always check for extraneous solutions in log equations. A valid algebraic answer might require log(negative) which is undefined.
Asymptotes & Holes — f(x) = p(x)/q(x)
Step-by-Step Process — always do in this order
① Factor fully → ② Cancel common factors → Holes → ③ Remaining denom zeros → VAs → ④ Compare degrees → HA or Slant
Domain = all x except zeros of the ORIGINAL denominator (before canceling). Holes are excluded even though they cancel out.
Finding Each Feature
vertical asymp.zeros of remaining denom after cancel
holefactors that cancel from both num & denom
hole y-coordplug x = a into simplified function
x-interceptszeros of numerator (after canceling holes)
y-interceptplug x = 0 into simplified form
domainℝ minus all original denom zeros
A zero in the numerator → x-intercept. A zero in the denominator (after cancel) → VA. A zero in both → hole. Never mix these up.
Horizontal & Slant Asymptotes
n < my = 0denom wins
n = my = (leading coeff ratio)aₙ/bₘ
n = m + 1slant asymptote (long divide)
n > m + 1no HA, no slant → ±∞
slant formy = quotient (ignore remainder)
Can a graph cross an asymptote? YES for HA (crosses in the middle, approaches at ±∞). NO for VA — never crosses a vertical asymptote.
Behavior Near Vertical Asymptotes
Sign Analysis Near VAs
methodpick test point on each side of VA
positive f →→ +∞
negative f →→ −∞
odd mult. VAopposite signs left & right → crosses axis
even mult. VAsame sign both sides → both go same direction
End behavior: as x → ±∞, only leading terms matter. Divide leading term of numerator by leading term of denominator.
Quick Reference: HA Degree Rules
deg(num) vs deg(denom)
n < my = 0
n = my = a/b   (lead coeffs)
n > mno HA (slant or none)
Slant asymptote ONLY when deg(num) = deg(denom) + 1 exactly. Use polynomial long division to find it.
Polar Fundamentals
Basics
point (r, θ)r = distance from pole, θ = angle
x = r cos θ
y = r sin θ
r = √(x² + y²)
θ = arctan(y/x)  (adjust for quadrant!)
same point(r, θ) = (−r, θ+π) = (r, θ+2π)
⚠ The quadrant of θ does NOT determine where the point is plotted. r can be negative. ALWAYS compute (x, y) = (r cos θ, r sin θ) to confirm the actual location.
Negative r — Critical Concept

When r < 0, the point is plotted in the opposite direction from θ — 180° away.


Example: (−3, π/4) plots in the same spot as (3, π/4 + π) = (3, 5π/4), which is in quadrant III — not quadrant I where π/4 points.

The curve still EXISTS when r is negative. Many MCQ traps rely on students assuming a negative r means the point doesn't exist in that region.
Increasing & Decreasing in Polar
The Key Insight — think rectangular
For r = f(θ), draw the rectangular graph with θ on the x-axis and r on the y-axis. This rectangular graph tells you everything.
rect. f(θ) rising→ r increasing → point moving away from pole
rect. f(θ) falling toward 0→ r decreasing → curve spiraling toward pole
rect. f(θ) = 0→ r = 0 → curve passes through the pole at that θ
rect. f(θ) goes below 0→ r negative → point flips direction, traced in opposite quadrant
rect. f(θ) at max→ farthest point from pole (tip of petal, outer loop)
one arch on rect. graph (0 → max → 0)→ one complete petal traced in polar
When asked "is r increasing?" — look at the rectangular graph. Is f(θ) going up? That's increasing r. Going toward zero? That's decreasing r. Dipping below zero? r is negative — still real, just flipped.

Symmetry & Intersections
Symmetry Rules
cos functionsymmetric about polar axis (x-axis)
sin functionsymmetric about θ = π/2 (y-axis)
replace θ → −θ, same eq.symmetric about polar axis
replace r → −r, same eq.symmetric about pole
replace θ → π−θ, same eq.symmetric about θ=π/2
Intersection Traps
Two polar curves can intersect at the same physical point with different (r, θ) pairs.
Always check: (1) set curves equal and solve, (2) check if both pass through the pole (r = 0 at different θ values still means they meet at the pole).

Polar Function Traps — Exam Watch List
High-Frequency Traps
⚠ Quadrant trapθ tells you the angle of the ray, but a negative r plots the point in the OPPOSITE direction. Always compute actual (x, y) before declaring the quadrant.
⚠ Increasing/decreasingThink about the rectangular graph of r = f(θ). If f is decreasing toward zero, r is decreasing (getting closer to pole). If f goes negative, r is negative — curve flips sides.
⚠ Inner loopTraced entirely when r < 0. Don't skip those θ intervals — the curve is still being drawn.
⚠ Rose petal countr = a cos(nθ): n odd → n petals; n even → 2n petals. Same rule applies with sin.
⚠ Same point, diff coordsA point can be represented multiple ways. (r, θ) = (−r, θ+π). Check all representations when finding intersections.
